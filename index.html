Logo Production Process

로고 제작, 이렇게 진행됩니다

고객과의 충분한 소통을 바탕으로, 단계적이고 체계적인 방식으로 진행합니다.

질문지 작성 → 시안 제안 (3일 이내) → 수정 (최대 2회) → 최종 확정 → 응용 이미지 제작
01
Direction Survey

로고 방향성 질문지 작성

로고 제작에 앞서 교회의 정체성과 방향성을 파악하기 위한 질문지를 작성합니다. 로고 형태, 교회 가치, 선호하는 분위기 등을 답변해 주시면 됩니다.

로고 형태 선택 교회 핵심 가치 선호·기피 분위기 참고 이미지 첨부
02
Design Proposal

로고 시안 제안

질문지 답변을 바탕으로 컨빌 디자인팀이 시안을 준비합니다. 답변 접수 후 3일 이내에 로고 시안을 제안해 드립니다.

3안 또는 4안 제공 교회 정체성 반영 목회 철학 기반 디자인
03
Revision

시안 검토 및 수정

제안된 시안을 교회와 함께 검토하고 피드백을 반영합니다. 수정은 최대 2회까지 제공됩니다.

1차 수정 2차 수정 (최종 조율)
04
Confirmation

최종 로고 확정

최종 로고를 확정하고 인쇄·디지털 환경 모두에서 사용 가능한 다양한 파일 형식으로 제공합니다.

AI / SVG 원본 PNG 투명 배경 JPG 배경 포함
05
Application Design

응용 이미지 제작

최종 로고 확정 후, 실제 사용 환경에 맞는 응용 이미지를 함께 제작합니다. 현수막·주보·SNS 등 다양한 매체에 적용한 결과물을 제공합니다.

현수막 적용 주보 / 인쇄물 SNS 프로필 간판 예시
✓
Summary

최종 결정사항 요약

컨빌디자인의 교회 로고 제작 서비스는 아래의 원칙으로 진행됩니다.

Service Commitment

  • 로고 제작 전 질문지 수집
  • HTML 폼으로 답변 접수
  • 답변 후 3일 이내 시안 제안
  • 로고 시안 3~4안 제공

  • 최종 확정 후 응용 이미지 제작
